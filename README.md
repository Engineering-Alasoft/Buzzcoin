# BUZ Crypto Landing Page

A modern, futuristic crypto token landing page built with React (JSX) and Tailwind CSS.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React 18 (JSX only, no TypeScript)
- Vite
- Tailwind CSS

## Project Structure

```
src/
├── App.jsx              # Main layout
├── main.jsx             # Entry point
├── index.css            # Tailwind + custom styles
└── components/
    ├── Navbar.jsx
    ├── Hero.jsx         # Includes CSS 3D coin visual
    ├── Stats.jsx
    ├── WhyBuzz.jsx
    ├── Tokenomics.jsx
    ├── Roadmap.jsx
    ├── FAQ.jsx
    └── Footer.jsx
```
